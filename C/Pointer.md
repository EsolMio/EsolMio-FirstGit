# 指针
- 指针定义：数据类型 *指针变量名 如：`int *pointer`。
- 指针为特殊类型，不能直接与普通类型的使用方式划等号。

## 初始化
指针的初始化特殊，如
`int *pointer = &a;`
```
int *pointer;
pointer=&a;
```
- 在定义时初始化的方式本质上和普通类型相同，但因为是声明所以必须要有"*"；
- 而在声明之后初始化则和普通类型相同，使用**变量名**。
- 初始化指针时声明类型是因为不同类型的存储长度不同，明确长度后才可从首地址正确读取数据。
